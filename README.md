# 🛡️ TechPulse 2026: Workforce Intelligence Dashboard

> **"Data is the ink; engineering is the pen. Together, they write the story of our economy."**

**TechPulse** is an interactive, SQL-powered analytics platform designed to decode the workforce volatility of 2026. This project bridges the gap between raw data and human narrative, providing a high-fidelity look at how AI is reshaping the global tech landscape.

## 🚀 The Vision
As a published author and software engineer, I believe data shouldn't just be viewed—it should be told. I built this dashboard to move beyond static spreadsheets and create a dynamic "Market Pulse" that quantifies the "AI Catalyst" factor in modern industry shifts.

## 🛠️ The Stack
- **Languages:** Python 3.13
- **Database:** SQL (SQLite3) 
- **Frontend:** Streamlit (Custom Dark Mode UI)
- **Visuals:** Plotly Express (Treemaps & Data Grains)

## 📂 Project Workflow
1. **The Engine:** `database_setup.py` automates the ETL process, migrating CSV data into a relational SQL environment.
2. **The Intelligence:** `app.py` serves the interactive layer, using dynamic SQL queries to filter market sectors in real-time.
3. **The Data:** Focused on 2025-2026 tech trends, specifically tracking AI-related transitions.
